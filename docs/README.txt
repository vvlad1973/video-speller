=================================================================
          Video Spell Checker - Portable EXE версия
=================================================================

Автоматическая проверка орфографии в видео
Распознавание текста и поиск ошибок на русском и английском языках

=================================================================
БЫСТРЫЙ СТАРТ
=================================================================

1. Запустите VideoSpellChecker.exe
2. Нажмите "Обзор..." и выберите видео файл
3. Настройте параметры (интервал, временной диапазон)
4. Нажмите "Начать проверку"
5. Дождитесь завершения обработки
6. Результаты откроются автоматически

=================================================================
СИСТЕМНЫЕ ТРЕБОВАНИЯ
=================================================================

- Windows 10/11 (64-bit)
- Минимум 4GB RAM (рекомендуется 8GB)
- ~2GB свободного места на диске для кэша моделей
- Права на запись в папку программы (для результатов)

=================================================================
ПАРАМЕТРЫ ОБРАБОТКИ
=================================================================

ИНТЕРВАЛ (секунды):
  - Определяет как часто извлекаются кадры из видео
  - По умолчанию: 2 секунды
  - Меньше значение = больше кадров = точнее, но медленнее
  - Больше значение = меньше кадров = быстрее, но можно пропустить ошибки

ВРЕМЕННОЙ ДИАПАЗОН:
  - Начало: с какого момента начать проверку (по умолчанию 00:00:00)
  - Конец: на каком моменте закончить (по умолчанию = конец видео)
  - Полезно для проверки только части видео

=================================================================
РЕЗУЛЬТАТЫ ОБРАБОТКИ
=================================================================

После обработки создаётся папка: screenshots_<имя_видео>_errors

В ней находятся:

  frame_XXX_errors.png - скриншоты кадров с ошибками
  frame_XXX_errors.txt - детали по каждому кадру:
    • Номер кадра и точное время (тайм-код)
    • Распознанный текст
    • Список найденных ошибок

  report-YYYYMMDD-HHMMSS.txt - итоговый отчёт:
    • Дата и время проверки
    • Параметры обработки
    • Статистика по всем кадрам
    • Детальная информация по ошибкам
    • Тайм-коды всех найденных ошибок

=================================================================
ПОЛЬЗОВАТЕЛЬСКИЙ СЛОВАРЬ
=================================================================

Вы можете добавить свои слова-исключения, которые не должны
помечаться как ошибки.

КАК ИСПОЛЬЗОВАТЬ:

1. Откройте меню "Файл" → "Открыть пользовательский словарь..."
2. Добавьте свои слова (по одному на строку)
3. Сохраните файл (Ctrl+S)
4. Перезапустите программу

ПРИМЕРЫ СЛОВ ДЛЯ ДОБАВЛЕНИЯ:

  # Технические термины
  PyTorch
  TensorFlow
  OpenCV

  # Названия брендов
  YouTube
  GitHub

  # Русские слова
  знакомиться
  информационные

РАСПОЛОЖЕНИЕ ФАЙЛА:
  custom_dictionary.txt (в той же папке, где VideoSpellChecker.exe)

=================================================================
СЛОВАРИ ДЛЯ ПРОВЕРКИ ОРФОГРАФИИ
=================================================================

Программа использует словари Hunspell (как в LibreOffice).

РАСПОЛОЖЕНИЕ:
  ./dictionaries/ru_RU.aff
  ./dictionaries/ru_RU.dic
  ./dictionaries/en_US.aff
  ./dictionaries/en_US.dic

АВТОМАТИЧЕСКАЯ ЗАГРУЗКА:
  Если словари отсутствуют, программа предложит их скачать
  при первом запуске.

=================================================================
ПЕРВЫЙ ЗАПУСК
=================================================================

При первом запуске программа:

1. Загрузит модели OCR (EasyOCR) - ~100MB
   • Загружается один раз
   • Сохраняется в папке пользователя
   • Последующие запуски будут быстрее

2. Может предложить скачать словари для проверки орфографии
   • Размер: ~5MB
   • Сохраняются в папке ./dictionaries/

Первый запуск может занять 2-5 минут в зависимости от
скорости интернета.

=================================================================
УСТРАНЕНИЕ ПРОБЛЕМ
=================================================================

ПРОГРАММА НЕ ЗАПУСКАЕТСЯ:
  • Убедитесь, что у вас Windows 10/11 (64-bit)
  • Проверьте антивирус - добавьте программу в исключения
  • Запустите от имени администратора

МЕДЛЕННАЯ РАБОТА:
  • Первый запуск всегда медленнее (загрузка моделей)
  • Увеличьте интервал между кадрами (например, до 5 секунд)
  • Обрабатывайте только нужный временной диапазон

СЛИШКОМ МНОГО ЛОЖНЫХ СРАБАТЫВАНИЙ:
  • Добавьте правильные слова в custom_dictionary.txt
  • OCR может неправильно распознавать некоторые символы

НЕ ХВАТАЕТ ПАМЯТИ:
  • Закройте другие программы
  • Обрабатывайте видео частями (используйте временной диапазон)

СЛОВАРИ НЕ ЗАГРУЖАЮТСЯ:
  • Проверьте подключение к интернету
  • Скачайте словари вручную с GitHub:
    github.com/LibreOffice/dictionaries
  • Положите файлы в папку ./dictionaries/

=================================================================
ТЕХНИЧЕСКИЕ ДЕТАЛИ
=================================================================

Используемые технологии:
  • EasyOCR - распознавание текста (русский/английский)
  • Hunspell - проверка орфографии (словари LibreOffice)
  • OpenCV - обработка видео
  • PyQt6 - графический интерфейс
  • PyTorch - движок для нейросетей OCR

Поддерживаемые форматы видео:
  MP4, AVI, MOV, MKV, FLV, WMV и другие форматы,
  поддерживаемые OpenCV
