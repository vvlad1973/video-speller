================================================================================
                        VIDEO SPELL CHECKER
                    Проверка орфографии в видео
================================================================================

ВЕРСИЯ ДЛЯ WINDOWS (ONE FILE)

================================================================================
1. УСТАНОВКА
================================================================================

1.1. Распакуйте ZIP-архив в любую удобную папку на вашем компьютере
     Например: C:\Programs\VideoSpellChecker\

1.2. После распаковки вы увидите следующие файлы:
     - VideoSpellChecker.exe  (основное приложение)
     - app.ico                (иконка приложения)
     - dictionaries/          (папка со словарями)
     - README.txt             (этот файл)

1.3. Готово! Приложение не требует установки и готово к работе.

================================================================================
2. ЗАПУСК ПРИЛОЖЕНИЯ
================================================================================

Дважды щелкните по файлу VideoSpellChecker.exe

ВАЖНО: При первом запуске Windows может показать предупреждение
"Windows защитила ваш компьютер". Это нормально для неподписанных приложений.

Чтобы запустить приложение:
- Нажмите "Подробнее"
- Нажмите "Выполнить в любом случае"

Или добавьте приложение в исключения Windows Defender.

================================================================================
3. ИСПОЛЬЗОВАНИЕ
================================================================================

3.1. Выберите видео файл
     - Нажмите кнопку "Обзор..." рядом с полем "Видео файл"
     - Выберите видео (поддерживаются: MP4, AVI, MKV, MOV и другие)

3.2. Выберите папку для результатов
     - Нажмите кнопку "Обзор..." рядом с полем "Папка вывода"
     - Выберите папку, куда сохранятся кадры с ошибками

3.3. Настройте параметры (опционально)
     - Интервал: каждые N секунд проверять кадр (по умолчанию: 2 сек)
     - Начало: с какого времени начать проверку (формат: ЧЧ:ММ:СС)
     - Конец: на каком времени закончить проверку

3.4. Нажмите "Начать проверку"

3.5. Дождитесь завершения обработки
     - Прогресс отображается в процентах
     - Текущий кадр показывается справа
     - Лог обработки выводится слева

================================================================================
4. РЕЗУЛЬТАТЫ
================================================================================

После завершения в папке вывода будут созданы:

4.1. Кадры с ошибками:
     - frame_1_errors.png, frame_2_errors.png, ... (PNG изображения кадров)
     - frame_1_errors.txt, frame_2_errors.txt, ... (списки ошибок для каждого кадра)

4.2. Итоговый отчёт:
     - report-YYYYMMDD-HHMMSS.txt (полный отчёт обработки)

Формат отчёта (report-*.txt):
    - Общая статистика обработки
    - Список всех найденных ошибок с отметками времени
    - Формат: Время | Слово с ошибкой | Предложенное исправление

================================================================================
5. ПОЛЬЗОВАТЕЛЬСКИЙ СЛОВАРЬ
================================================================================

Вы можете добавить свои слова в словарь, чтобы они не считались ошибками:

5.1. В главном окне выберите меню "Файл" → "Открыть пользовательский словарь"

5.2. Откроется файл custom_dictionary.txt

5.3. Добавьте свои слова (одно слово на строку)
     Например:
     YouTube
     livestream
     NFT

5.4. Сохраните файл и перезапустите проверку

================================================================================
6. НАСТРОЙКИ
================================================================================

Приложение сохраняет настройки в файле config.ini:
- Последняя папка вывода

Настройки загружаются автоматически при следующем запуске.

ПРИМЕЧАНИЕ: Остальные параметры (видео файл, интервал, время начала/конца)
сохраняются только во время сессии и сбрасываются при перезапуске приложения.

================================================================================
7. СИСТЕМНЫЕ ТРЕБОВАНИЯ
================================================================================

Минимальные:
- Windows 10/11 (64-bit)
- 4 GB RAM
- 2 GB свободного места на диске
- Процессор с поддержкой AVX (Intel/AMD 2011 года и новее)

Рекомендуемые:
- 8 GB RAM или более
- SSD диск
- Многоядерный процессор

================================================================================
8. РЕШЕНИЕ ПРОБЛЕМ
================================================================================

Проблема: Приложение не запускается
Решение:
  - Убедитесь, что у вас Windows 10/11 64-bit
  - Попробуйте запустить от имени администратора
  - Проверьте, что приложение не заблокировано антивирусом

Проблема: "VCRUNTIME140.dll не найден" или подобные ошибки
Решение:
  - Установите Microsoft Visual C++ Redistributable 2015-2022
  - Скачать: https://aka.ms/vs/17/release/vc_redist.x64.exe

Проблема: Приложение медленно работает
Решение:
  - Увеличьте интервал проверки (например, до 5 секунд)
  - Закройте другие программы для освобождения RAM
  - Используйте SSD вместо HDD для папки вывода

Проблема: Не распознается текст на видео
Решение:
  - Проверьте, что текст достаточно крупный и четкий
  - Убедитесь, что текст на русском или английском языке
  - Попробуйте уменьшить интервал для более частой проверки

================================================================================
9. О ПРИЛОЖЕНИИ
================================================================================

Video Spell Checker - приложение для автоматической проверки орфографии
в видеороликах с использованием технологий:

- EasyOCR - распознавание текста на кадрах
- Hunspell - проверка орфографии (русский/английский)
- PyQt6 - графический интерфейс
