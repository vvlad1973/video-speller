1️⃣ БЫСТРЫЙ СТАРТ
═════════════════

Для ONE FILE версии:
--------------------
1. Запустите приложение:
   ./VideoSpellChecker

2. Выберите видео файл через GUI
3. Нажмите "Начать проверку"
4. Дождитесь завершения обработки
5. Результаты откроются автоматически

Для ONE DIR версии:
-------------------
1. Перейдите в папку VideoSpellChecker:
   cd VideoSpellChecker

2. Запустите приложение:
   ./VideoSpellChecker

2️⃣ СИСТЕМНЫЕ ТРЕБОВАНИЯ
═════════════════════════

✓ Linux (Ubuntu 20.04+, Fedora 34+, или аналог)
✓ 64-bit архитектура (x86_64)
✓ Минимум 4GB RAM (рекомендуется 8GB)
✓ 2GB свободного места на диске
✓ Графическая среда (X11 или Wayland)

Библиотеки (обычно уже установлены):
- libGL.so.1 (OpenGL)
- libxcb
- libfontconfig
- libfreetype

Если GUI не запускается, установите:
sudo apt install libxcb-xinerama0 libxcb-cursor0  # Debian/Ubuntu
sudo dnf install libxcb xcb-util-cursor           # Fedora

3️⃣ УСТАНОВКА
═════════════

Опция A: Использование .desktop файла
--------------------------------------
1. Скопируйте приложение в /opt:
   sudo cp -r VideoSpellChecker /opt/

2. Отредактируйте пути в VideoSpellChecker.desktop:
   nano VideoSpellChecker.desktop

   Измените Exec и Icon на:
   Exec=/opt/VideoSpellChecker/VideoSpellChecker
   Icon=/opt/VideoSpellChecker/app.ico

3. Установите .desktop файл:
   sudo cp VideoSpellChecker.desktop /usr/share/applications/

4. Обновите кэш меню:
   sudo update-desktop-database

Теперь приложение появится в меню приложений!

Опция B: Portable версия
-------------------------
Просто запускайте из любой папки:
./VideoSpellChecker

4️⃣ ИСПОЛЬЗОВАНИЕ
═════════════════

Графический интерфейс:
----------------------
1. Запустите: ./VideoSpellChecker

2. В окне приложения:
   - Нажмите "Обзор..." чтобы выбрать видео
   - Установите интервал между кадрами (по умолчанию 2 сек)
   - Опционально: укажите временной диапазон для анализа
   - Выберите папку для результатов (по умолчанию: ./out)
   - Нажмите "Начать проверку"

3. Наблюдайте за прогрессом в реальном времени

4. После завершения папка с результатами откроется автоматически

Консольная версия:
------------------
Для запуска из консоли используйте Python версию:
python3 gui.py

5️⃣ ПАРАМЕТРЫ
═════════════

Интервал между кадрами:
- 1-2 секунды: максимальная точность (медленнее)
- 2-3 секунды: баланс скорости и точности (рекомендуется)
- 5+ секунд: быстрая проверка (может пропустить ошибки)

Временной диапазон:
- Начало [час:мин:сек]: с какого момента начать анализ
- Конец [час:мин:сек]: где закончить анализ
- Оставьте 00:00:00 чтобы обработать всё видео

Папка вывода:
- По умолчанию: <каталог_приложения>/out
- Можно выбрать любую папку
- Настройка сохраняется между запусками

6️⃣ РЕЗУЛЬТАТЫ
══════════════

После обработки будет создана папка:
<папка_вывода>/screenshots_<имя_видео>_errors/

Содержимое:
-----------
📄 frame_XXX_errors.png
   Скриншоты кадров где найдены ошибки

📝 frame_XXX_errors.txt
   Детали по каждому кадру:
   - Номер кадра и точный тайм-код
   - Полный распознанный текст
   - Список найденных орфографических ошибок

📋 report-YYYYMMDD-HHMMSS.txt
   Итоговый отчёт о проверке:
   - Дата и время проверки
   - Параметры обработки
   - Статистика по всем кадрам
   - Тайм-коды всех ошибок

При повторной обработке:
- Старые скриншоты и .txt файлы удаляются
- Отчёты сохраняются (история проверок)

7️⃣ ПОЛЬЗОВАТЕЛЬСКИЙ СЛОВАРЬ

Файл: custom_dictionary.txt (в той же папке с приложением)

Добавление слов:
----------------
1. Откройте файл в текстовом редакторе:
   nano custom_dictionary.txt

2. Добавьте слова (по одному на строку):
   Python
   GitHub
   Docker
   название_проекта

3. Сохраните файл (Ctrl+O, Enter, Ctrl+X в nano)

4. Перезапустите приложение

Или через GUI:
--------------
Меню → Файл → Открыть пользовательский словарь...

Примеры слов для добавления:
- Технические термины: API, JSON, HTTP
- Бренды: YouTube, Google, Microsoft
- Аббревиатуры: OCR, ML, AI
- Имена собственные

8️⃣ УСТРАНЕНИЕ НЕПОЛАДОК
═════════════════════════

Проблема: Приложение не запускается (ошибка библиотек)
-------------------------------------------------------
Решение:
sudo apt install libxcb-xinerama0 libxcb-cursor0 libgl1  # Ubuntu/Debian
sudo dnf install libxcb xcb-util-cursor mesa-libGL       # Fedora

Проблема: "Permission denied"
-----------------------------
Причина: Скрипт сборки не установил права на выполнение

Решение:
chmod +x VideoSpellChecker

Проблема: Не создаются скриншоты
---------------------------------
Если возникла проблема, проверьте:

1. Права доступа к папке вывода:
   ls -la /путь/к/папке/вывода

2. Наличие свободного места:
   df -h

3. Корректность пути к видео файлу

Проблема: Медленная работа
--------------------------
Решение:
- Увеличьте интервал между кадрами (3-5 сек)
- Закройте другие приложения
- Используйте более короткий временной диапазон

Проблема: Слишком много ложных срабатываний
--------------------------------------------
Решение:
Добавьте правильные слова в custom_dictionary.txt

Проблема: Нет звука при открытии результатов
---------------------------------------------
Это нормально - программа только проверяет текст на видео,
звук не используется.
