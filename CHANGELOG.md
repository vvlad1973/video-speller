# История изменений

## Реструктуризация проекта (08.11.2025)

### Новая структура каталогов

```bash
video-speller/
├── main.py                     # Точка входа (главный файл)
├── assets/                     # Ресурсы (иконки, изображения)
│   ├── app.ico
│   └── splash.png
├── src/                        # Модули приложения
│   ├── __init__.py
│   ├── video_speller.py       # Логика проверки орфографии
│   ├── gui_widgets.py         # Виджеты GUI
│   ├── main_window.py         # Сгенерированный код UI
│   ├── about_dialog.py        # Диалог "О программе"
│   └── download_dictionaries.py  # Утилита загрузки словарей
├── ui/                         # UI файлы Qt Designer
│   ├── main_window.ui
│   └── about_dialog.ui
├── docs/                       # Документация для пользователей
│   ├── README.txt             # Для Windows .exe
│   ├── README_LINUX.txt       # Для Linux сборок
│   └── QT_DESIGNER.md         # Редактирование UI
├── dictionaries/               # Словари Hunspell
├── custom_dictionary.txt       # Пользовательский словарь
├── requirements.txt            # Зависимости Python
├── README.md                   # Основная документация (GitHub)
├── build_one_file.bat/sh      # Скрипты сборки (один файл)
└── build_dir.bat/sh           # Скрипты сборки (директория)
```
